Linux CentOS 7 iptables 开放端口

2019-01-09 | 阅读 | 286 字 | 1 分钟

查看已开放端口:

iptables -L

添加端口入口允许(示例：3306)：

iptables -I INPUT -p tcp —dport 3306 -j ACCEPT

添加端口出口允许（示例：3306）:

iptables -I OUTPUT -p tcp —sport 3306 -j ACCEPT

保存:

service iptables save

查看打开的端口:

iptables status

重启:

service iptables restart

参数详解

-t<表>：指定要操纵的表；
-A：向规则链中添加条目；
-D：从规则链中删除条目；
-i：向规则链中插入条目；
-R：替换规则链中的条目；
-L：显示规则链中已有的条目；
-F：清楚规则链中已有的条目；
-Z：清空规则链中的数据包计算器和字节计数器；
-N：创建新的用户自定义规则链；
-P：定义规则链中的默认目标；
-h：显示帮助信息；
-p：指定要匹配的数据包协议类型；
-s：指定要匹配的数据包源ip地址；
-j<目标>：指定要跳转的目标；
-i<网络接口>：指定数据包进入本机的网络接口；
-o<网络接口>：指定数据包要离开本机所使用的网络接口。
